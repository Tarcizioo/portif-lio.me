## Projeto - Portal Animes V2

**Título:** Portal Animes V2 - Plataforma social PWA com React, GraphQL e Firebase

Desenvolvi uma plataforma social completa para descobrir, acompanhar e organizar animes. A aplicação integra a AniList por GraphQL e utiliza Firebase Authentication e Cloud Firestore para autenticação, persistência em tempo real e recursos sociais.

Principais entregas:

- Catálogo com busca, filtros persistentes, paginação infinita, calendário e páginas detalhadas.
- Biblioteca pessoal com progresso, importação/exportação, estatísticas e recomendações.
- Perfis públicos/privados, comentários, seguidores, notificações e comparação de bibliotecas.
- PWA instalável, cache persistente de 24 horas, rotas carregadas sob demanda e suporte a acessibilidade.
- Pipeline no GitHub Actions com lint, build e 47 testes automatizados, incluindo regras de segurança do Firestore.

**Tecnologias:** React 19, Vite 7, React Router, AniList GraphQL, Firebase Auth, Cloud Firestore, TanStack Query, Tailwind CSS, Vitest, Testing Library, PWA e GitHub Actions.

**Aplicação:** https://portal-animes-v2.vercel.app/  
**Código:** https://github.com/Tarcizioo/portal-animes-V2

## Projeto - Voxel Terrain Engine

**Título:** Voxel Terrain Engine - Computação gráfica com Three.js e WebGL

Experimento de renderização voxel 3D em tempo real no navegador, criado para estudar computação gráfica, otimização de renderização e lógica de interação.

- Geração procedural de terreno por função matemática de altura.
- Renderização de milhares de blocos com THREE.InstancedMesh.
- Controles em primeira pessoa, gravidade, salto e colisão básica.
- Raycasting para detectar e remover blocos visualmente.

**Tecnologias:** JavaScript, Three.js, WebGL, HTML e CSS.

**Demo:** https://tarcizioo.github.io/Voxel-Terrain-Engine/  
**Código:** https://github.com/Tarcizioo/Voxel-Terrain-Engine
